# Contributing to Image-to-Image Translator 🚀

First of all, thank you for considering contributing to this project! This tool aims to revolutionize the localization workflow in the publishing industry through AI[cite: 7].

## 🛠️ Required Skills & Onboarding
To maintain the technical standards of this project, contributors should possess or be willing to learn the following skills:

*   **Python 3.8+**: Deep understanding of multi-threading (to handle UI and heavy processing)[cite: 7].
*   **Computer Vision (OpenCV)**: Familiarity with adaptive thresholding and contour analysis for image pre-processing[cite: 7].
*   **Deep Learning (EasyOCR/PyTorch)**: Knowledge of how OCR engines detect and recognize text in diverse layouts[cite: 7].
*   **Version Control**: Proficiency in Git for branching, merging, and managing pull requests.

## 📝 Contribution Process
We follow a structured workflow to ensure software stability:

1.  **Fork the Project**: Create your own copy of the repository.
2.  **Create a Feature Branch**: Use clear names like `feature/new-filter` or `bugfix/ocr-accuracy`.
3.  **Follow Coding Standards**: 
    *   Use **Google-style Docstrings** for all new functions.
    *   Maintain the current modular architecture (`gui.py`, `logic.py`, etc.).
4.  **Update Documentation**: If you add a new feature, update the technical documentation using `pdoc`.
5.  **Submit a Pull Request (PR)**: Describe the strategic value of your change and how it improves the digital transformation of the translation process.
